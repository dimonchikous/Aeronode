# Аэронода - твой помощник в отслеживании комнатного воздуха.
[English](README.md) | [Русский](README.ru.md)

[Инструкция](TUTORIAL.ru.md)

## Основная задумка.
Воздух в квартире редко отслеживается, чаще всего проверяется посредственно и на основе своих ощущений. Это может часто вредить здоровью человека.

Этот проект предназначен для решения этой проблемы, то есть отслеживания состояния воздуха в комнате. Также при подключении устройства к серверу он сможет отправлять систематические уведомления о погоде в городе пользователя, уведомлять о качестве воздуха, а также импровизированную сигнализацию при критических показаниях.

## Принцип работы.

### Оффлайн
При **оффлайн** режиме устройство самостоятельно отображает информацию с помощью oled-экрана.

Само устройство на базе ESP32-WROOM32 devkit v1.

Также помимо этого используется:
* SCD-40 для измерения уровня CO2.
* BME280 для измерения температуры, давления, влажности воздуха.
* OLED-экран SSD1306 128x64 - для вывода информации.

### Онлайн
![Схема работы при онлайне](img/RU/roadmap.png)

Раз в минуту устройство посылает json пакет на сервер с телеграм ботом. Бот отслеживает эти пакеты и при запросе пользователя отправляет данные, что позволяет просматривать показания вне дома. Также есть возможность устанавливать расписание показаний, возможность просмотра погоды, а также возможность включения имрповизированной сигнализации, которая срабатывает на большие температуры и/или кол-во CO2. Главное чтобы у вас был сервер или завалявшийся ненужный компьютер.

В качестве мессенджера для бота используется телеграм.
